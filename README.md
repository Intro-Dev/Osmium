# Osmium

## Basics
Osmium is a quality of life mod for fabric 1.17 and above.

## Features
Features include
- ToggleSprint
- ToggleSneak
- Fullbright
- Gui editing
- NoRain
- Fps display
- LowFire
- NoHurtBob
- Optifine cape renderer
